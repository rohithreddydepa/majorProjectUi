Analysis

  • With the increase in online education, the number of platforms supporting question answer forums are increasing. Out of platforms like Stack Overflow, Coursera, Quora, Stack Overflow is one of the most popular online question-answer forums and there is a huge amount of information in this forum. So, to make the system efficient, it is very necessary to detect the class to which a particular piece of information belongs to. This detection is called tagging. It helps in easy retrieval and organization of data. Our project aims at developing a system for autonomously detecting the tags of Stack Overflow questions using NLP and Machine Learning.

  • The size of the data is a hindrance to improve the system. Currently, websites like Quora are expecting users to manually enter the tags related to their queries. But, manual tagging is a difficulty for the end user and causing poor user experience. This will benefit both the questioning and answering people.Answering people can choose the topic which they interset they can filter out the irrelevant question .And Question people can get the answers fast if tag is predict
